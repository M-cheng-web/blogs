本地端口操作 #

  • 查看: sudo lsof -i:8080
  • 终止: sudo kill -9 <pid>

ng #

  • 启动: sudo nginx
  • 重启: sudo nginx -s reload
  • 停止: sudo nginx -s stop
  • 打开配置地址: 控制台 nginx -h, 然后command + 最后一位地址
